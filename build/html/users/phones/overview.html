Les téléphones dans RaspiSMS¶

RaspiSMS vous permet de créer un ou plusieurs téléphones par utilisateur. Lors de l’envoi d’un SMS, vous pourrez définir si vous souhaiter utiliser un téléphone en particulier, ou bien si vous souhaitez laisser RaspiSMS choisir un téléphone au hasard.
